# Base Styles

기본 스타일 폴더입니다. 다른 모든 스타일에 의존하는 기본 요소들을 포함합니다.

## 포함될 파일들

### 1. variables.css
- CSS 변수 정의 (색상, 간격, 폰트, 그림자 등)
- `:root` 선택자 내의 모든 변수들

### 2. reset.css
- 기본 CSS 리셋
- `*`, `body`, `#app` 등 기본 요소 스타일

### 3. typography.css
- 기본 타이포그래피 스타일
- 폰트 패밀리, 기본 텍스트 스타일

## 의존성
- **가장 기본**: 다른 모든 스타일에 의존
- **독립적**: 다른 파일에 의존하지 않음

## 사용 순서
1. variables.css (CSS 변수)
2. reset.css (기본 리셋)
3. typography.css (타이포그래피)
