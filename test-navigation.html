🚀 하단 네비게이션 통일 테스트

모든 탭에서 하단 네비게이션이 동일하게 표시되는지 확인합니다.

✅ 통일된 디자인 요소들

• 탭 버튼 높이: 80px (고정)

• 아이콘 크기: 24px (고정)

• 레이블 크기: 11px (고정)

• 패딩: 8px 4px (고정)

• 활성 탭: 상단 파란색 보더 + 연한 파란색 배경

🎯 테스트 방법

1. 하단의 5개 탭을 순서대로 클릭해보세요

2. 각 탭 전환 시 네비게이션이 흔들리지 않는지 확인

3. 모든 탭에서 동일한 높이와 크기인지 확인

4. 활성/비활성 상태가 일관되게 표시되는지 확인

📱 반응형 테스트

• 모바일 뷰에서 테스트 (브라우저 개발자 도구 사용)

• 다양한 화면 크기에서 네비게이션이 일관되게 표시되는지 확인
